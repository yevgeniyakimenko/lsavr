FAQ

When NOT to use LinkSaver?

  • When you need to share sensitive information. Although the database uses cryptographic encryption to store your data, there is no guarantee against data loss, hacking or extortion. Use this service at your own risk.
  • When you are using a VPN service. Your endpoint IP address may be shared with an unknown number of other VPN users.
  • If your Internet Service Provider changes your public IP address. Once your public IP address changes, you will no longer be able to see your saved links. For this reason, it is probably a good idea to delete all your links if you are not planning on using them in the near future.
  • If you need to share more than 50 links at once. LinkSaver is for personal, small-scale use only. If you need to share larger amounts of data, please consider using alternative services.

How does it work?

Usually, devices connected to the same WiFi or LAN home network share the same external (public) IP address. This allows LinkSaver to show the same links to all users whose network requests come from the same public IP address.
